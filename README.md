crouter
=======

Custom based ip router

Install
======
```
sudo apt-get install libpcap0.8-dev
```

Alias to add in ~/.bashrc or ~/.profile (Shorcuts)
=================================================
```
alias m = 'make build'
alias r = ''
```

Build
=====
```
>> m
```

Run
==
```
>> r
```
